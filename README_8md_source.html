3 ## Software for Large-scale Bound-constrained Optimization
4 [`L-BFGS-B`](http://users.iems.northwestern.edu/~nocedal/lbfgsb.html) is a limited-memory quasi-Newton code for bound-constrained optimization,
5 i.e., for problems where the only constraints are of the form `l <= x <= u`.
6 It is intended for problems in which information on the Hessian matrix is difficult to obtain,
7 or for large dense problems.
8 `L-BFGS-B` can also be used for unconstrained problems, and in this case performs similarly to its predecessor,
9 algorithm [`L-BFGS`](http://users.iems.northwestern.edu/~nocedal/lbfgs.html) (Harwell routine VA15). The algorithm is implemented in Fortran 77.

27 * C. Voglis and I. E. Lagaris, [BOXCQP: An Algorithm for Bound Constrained Convex Quadratic Problems](http://www.cs.uoi.gr/~voglis/boxcqp.pdf) (2004), 1st International Conference "From Scientific Computing to Computational Engineering", Athens, Greece

51 ## Notes on this repository
